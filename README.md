# Stable Fluids (Web)

Jos Stam の論文 *Stable Fluids (2001)* の手法（拡散・移流・圧力投影）をベースに、ブラウザ上で動く 2D 流体シミュレーションを実装したサンプルです。

## 使い方

`index.html` をブラウザで開くだけで動作します。

- ドラッグ: 流体へ力と染料を注入
- `R` キー: 状態をリセット
- UI の「シミュレーション細かさ」: 解像度を標準 (`1/1`)・軽量 (`1/4`)・超軽量 (`1/16`) から選択
- ソルバ反復回数: FPS を見ながら自動で調整（HUD に現在値を表示）
- UI の「AA 強度」: 手動調整可能。ボタンで推奨値にリセット
- UI の「移動補完」: マウス移動補完をベジェ／直線から選択
- UI の表示切り替え: 染料（密度）／速度場ベクトル／圧力／渦度を ON/OFF
- UI の「染料ブラシサイズ(px)」: ドラッグ時に注入する染料の半径を調整
- UI の「クリック時の力の強さ」「クリック時の染料の量」: クリック時に発生するバーストの強さを調整
- ウィンドウリサイズ: 自動でキャンバス／グリッドを再生成

## ローカルサーバーで実行（任意）

```bash
python3 -m http.server 4173
```

その後 `http://localhost:4173` を開いてください。
